## Project Overview
This project is part of the Digital Literacy course at VIT Bhopal University. The objective of this project is to develop essential digital skills required for academic and professional growth.

As a Student Digital Ambassador, this project focuses on helping peers understand digital tools, build professional profiles, communicate effectively, and stay safe online.

The project is divided into five tasks, each covering a different module of digital literacy.

## Repository Structure

report/  
Contains the final project report in PDF or DOCX format.

task-1-presentation/  
Contains the infographic or presentation created using tools like Canva or Prezi along with a brief description.

task-2-portfolio/  
Contains screenshots of professional profiles such as GitHub, LinkedIn, or other platforms and reflection notes.

task-3-platforms/  
Contains screenshots of coding platform activity and Google Form along with proof of participation.

task-4-email-etiquette/  
Contains sample professional email drafts and a social media etiquette checklist.

task-5-cybercrime/  
Contains a cybercrime case study and a prevention checklist.

## Task Summaries

Task 1: Digital Literacy Infographic  
Created a visual infographic explaining digital literacy concepts, tools, and safe internet practices using an online design tool.

Task 2: Digital Portfolio  
Developed professional profiles on platforms such as GitHub and LinkedIn and learned their importance for career growth.

Task 3: Platforms Exploration  
Explored coding platforms and completed beginner challenges. Also created a Google Form for digital literacy awareness.

Task 4: Email Etiquette  
Drafted professional emails and created a checklist for responsible communication and social media usage.

Task 5: Cybercrime Awareness  
Researched cybercrime types and created a case study along with a prevention checklist to promote online safety.
